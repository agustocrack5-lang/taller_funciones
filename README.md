Descripción general
Este repositorio contiene la solución del taller de funciones en C++, donde se trabajan conceptos como paso por valor, referencias, punteros y manejo de arreglos.


Archivos:

bloque0_warmup.cpp
Ejercicios de diagnóstico sobre paso por valor y por referencia. Incluye predicciones del comportamiento del código antes de ejecutarlo.

bloque1_referencias.cpp
Implementación de funciones usando referencias (&). Se demuestra cómo una función puede modificar directamente variables originales.

bloque2_punteros.cpp
Ejercicios con punteros (*), incluyendo acceso a direcciones de memoria, desreferenciación y comparación entre punteros y referencias.

bloque3_desafio.cpp
Programa final que integra todos los conceptos:
- Manejo de arreglos
- Uso de punteros para modificar datos
- Uso de referencias para retornar valores
- Cálculo de promedio y ajuste de notas


Conclusión:
Este taller permitió entender la diferencia entre paso por valor, referencia y punteros, y cómo se utilizan en funciones para modificar datos de forma eficiente.
